Stuff I've Done

SensorConnect

SensorConnect is the desktop software for Lord Sensing's Wireless and Inertial product lines. It allows customers to configure their hardware and visualize massive amounts of data in real-time.

SensorConnect uses the Chromium Embedded Framework. I work on both the frontend (HTML, CSS, JavaScript) and backend (C++) with a team of developers. The use of WebGL, combined with smart data algorithms, provides the user with a smooth data visualization experience.

MSCL

I am the lead developer on MSCL (MicroStrain Communication Library). It makes it simple to write code to interact with LORD Sensing's Wireless and Inertial sensors.

The goal of MSCL is to have a library that is up to date, cross platform, multi-language, and well tested. MSCL is written in C++, with libraries also available for Python and .NET (C#, VB, MATLAB, LabVIEW) by using SWIG. It is fully documented, unit tested, and includes example code and quick start guides.

LORD uses MSCL internally in its own projects, such as SensorConnect, and the WSDA-1500. It is available open source on GitHub.
